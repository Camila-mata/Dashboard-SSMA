# 📊 Dashboard de Análise de Performance de SSMA  
**Transformando dados de segurança em decisões estratégicas**

---

## 📌 Contexto

A gestão de **Saúde, Segurança e Meio Ambiente (SSMA)** gera um grande volume de dados operacionais diariamente. No entanto, dados isolados em planilhas não geram impacto — é a **análise estruturada que permite prevenir riscos e orientar decisões**.

Este projeto foi desenvolvido com o objetivo de demonstrar como a utilização de ferramentas de **Business Intelligence (BI)** pode transformar dados operacionais em **insights acionáveis**, contribuindo para uma cultura de prevenção e melhoria contínua.

---

## 🎯 Objetivo do Projeto

Construir um dashboard interativo capaz de:

- Centralizar indicadores críticos de segurança  
- Facilitar o monitoramento de performance operacional  
- Apoiar a tomada de decisão baseada em dados  
- Identificar padrões e possíveis riscos  

---

## ⚠️ Observação sobre os dados

Para fins de confidencialidade, a base de dados foi **totalmente anonimizada e simulada**, não representando cenários reais.

## 📊 Principais Indicadores Monitorados

- Taxa de incidentes  
- Indicadores de segurança operacional  
- Distribuição de ocorrências por área  
- Tendências ao longo do tempo  
- Comparação entre períodos  

---

## 📈 Principais Insights Gerados

- Identificação de **variações na taxa de incidentes ao longo do tempo**, permitindo análise de tendências  
- Comparação entre áreas evidenciando **setores com maior exposição a riscos**  
- Monitoramento contínuo de indicadores possibilitando **antecipação de cenários críticos**  
- Visualização consolidada facilitando a identificação de **padrões operacionais**  

---

## 🎯 Possíveis Tomadas de Decisão

Com base nos dados apresentados, um gestor poderia:

- Direcionar treinamentos para áreas com maior índice de incidentes  
- Reforçar práticas de segurança em períodos de maior risco  
- Priorizar inspeções e auditorias em setores críticos  
- Acompanhar a efetividade de ações preventivas ao longo do tempo  

---

## 🧠 Valor do Projeto

Este dashboard demonstra como a análise de dados pode evoluir de um nível operacional para um nível estratégico, contribuindo para:

- Redução de riscos operacionais  
- Prevenção de acidentes  
- Melhoria contínua dos processos de segurança  
- Tomada de decisão mais rápida e assertiva

## 🛠️ Tecnologias Utilizadas
